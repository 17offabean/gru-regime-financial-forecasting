# GRU Regime-Dependent Financial Forecasting

This repository contains the complete implementation for the Machine Learning project report titled:

**“Regime-Dependent Performance of a GRU Model for Financial Time Series Forecasting.”**

## Contents
- `GRU_Regime_Dependent_Financial_Forecasting.ipynb`  
  A single Jupyter Notebook containing the full pipeline, including:
  - Feature engineering
  - GRU model architecture and training
  - Trading strategy and evaluation
  - Monte Carlo permutation testing

## Notes
The code is provided to support transparency and reproducibility. All methodological details, experimental design, and results are fully described in the accompanying report.
